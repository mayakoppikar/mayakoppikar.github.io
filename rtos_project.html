RTOS for Formula 1 Robot

This project focuses on building a real-time operating system (RTOS) for a Formula 1 robot. The RTOS manages motor control, sensor input, and inter-task communication with strict timing constraints.

Key Features
  • Task scheduling with priority-based preemption
  • Motor control via PWM
  • Sensor fusion from LIDARs
  • Interfacing via CAN bus
Tech Stack
  • Programming: C, ARM Assembly
  • Hardware: STM32 Microcontroller
  • RTOS Concepts: Scheduling, Semaphores, Queues
Design Process
  1. OS Kernel
  2. Memory Management
  3. Sensor Data Acquisition
  4. Motor Control and Race Strategy
  5. Testing & Tuning
Major Challenges & Key Takeaways
  • Integrating multiple sensor inputs in real-time without missing deadlines
  • Debugging timing issues in preemptive task scheduling
  • Ensuring reliable motor control with PWM under varying loads
  • Key Takeaways:
    • RTOS design requires careful planning of task priorities
    • Simulations and incremental testing greatly reduce integration errors
    • Documenting design decisions helps in troubleshooting and future upgrades
