RTOS for Formula 1 Robot

This project focuses on building a real-time operating system (RTOS) for a Formula 1 robot. The RTOS manages motor control, sensor input, and inter-module communication with strict timing constraints.

Key Features

  • Task scheduling with priority-based preemption
  • Motor control via PWM with interrupt-driven feedback
  • Sensor fusion from IMU and encoders
  • Interfacing via UART and CAN bus

Tech Stack

  • Programming: C, ARM Assembly
  • Hardware: STM32 Microcontroller
  • RTOS Concepts: Scheduling, Semaphores, Queues
