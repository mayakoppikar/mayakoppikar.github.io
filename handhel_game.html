Original Handheld Embedded Systems Game

This project involved designing and building a custom handheld game using embedded systems. The system uses a microcontroller, input buttons, and an LCD display to provide an interactive gaming experience.

Key Features

  • Custom game logic implemented in C on a microcontroller
  • Interactive input using push buttons
  • Output displayed on an LCD screen
  • Low-power design for handheld portability
