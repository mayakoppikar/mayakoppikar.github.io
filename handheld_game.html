Longhorn Connections: Handheld Embedded Systems Game

This project, called Longhorn Connections, is a re-creation of the NYT Connections, our favorite of the NYT Games set! I worked on this project with one lab partner as part of an undergraduate Embedded Systems Introduction class. This project was took us around 2 months to complete (April 2024 - May 2024).


Key Features
  • Analog to Digital Conversion (ADCs), Digital to Analog Conversion (DACs)
  • Buttons, Potentiometers, LEDs, LCD Display
  • Embedded Concepts: Timers, Interrupts, PWM
Tech Stack
  • Programming: C, C++
  • Hardware: TI MSPM0+ Microcontroller
  • Environment: Code Composer Studio (CCS)


Design Process
  1. MCU Peripherals: We worked on drivers for several MSPM0 peripherals including: GPIO: Added LEDs to indicate game status ADCs: Analog to digital conversion to incorporate digitized sounds and to control potentiometer for mode selection UART: Allows us to read in data from terminal

  2. Mapping Functions: Our game requires a key-value pair mapping between two connected words. The map library was too large to include in our code, so we implemented a map struct from scratch to represent these pairings and wrote algorithms to iterate through the pairs.

Major Challenges & Key Takeaways
  • Key Takeaways + Skils:
    • Communication protocols: UART
    • Interfacing hardware components in embedded systems
